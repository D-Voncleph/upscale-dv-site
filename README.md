# UPSCALE DV

> Web & App Development Agency — [upscaledv.com](https://upscaledv.com)

Custom web applications, mobile apps, and cloud-native systems for businesses that refuse to stay small.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React 19, TypeScript, Tailwind CSS 4
- **Animations:** Framer Motion
- **Forms:** React Hook Form + Zod
- **Deployment:** Vercel (auto-deploy on push to main)
- **Contact:** n8n webhook integration

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Push to `main` triggers automatic deployment on Vercel.
